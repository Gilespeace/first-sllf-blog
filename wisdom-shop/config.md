# 项目配置

## 项目 AppID 配置

1. **组长**(管理员)登录自己微信小程序后台
2. **小组成员**提供自己的微信号给组长

组长添加组员为**项目成员**，这样组员才能使用组长填入的 AppID 开发小程序
